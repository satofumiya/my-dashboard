# My Dashboard

Vpassのクレカ利用通知メールを自動解析し、月々の支出を見える化する個人向け家計ダッシュボードです。

## 機能

- **クレカ利用履歴** — Vpassからのメールを自動取得・一覧表示
- **カテゴリ別集計** — 利用先ごとにカテゴリを設定し、カテゴリ別の支出を表示
- **支払い見込み** — 固定費（サブスク・固定費）+ 変動費（クレカ実績）で今月の支払い見込みを表示
  - クレカ払いのサブスクは実際に請求されると「請求済 ✓」に移動し、二重計上を防ぐ
- **サブスク・固定費管理** — 月額/年額の支払いを一覧管理、年1回の支払いは該当月のみ表示
- **予算管理** — カテゴリ別の予算と実績を比較
- **予定・タスク** — Googleカレンダー連携、簡易タスク管理
- **冷蔵庫管理** — 食材の在庫管理
- **レシピ管理** — 食材から作れるレシピ一覧

## 構成

```
dashboard/
├── gas/api.gs       ← バックエンド（Google Apps Script）
├── src/             ← フロントエンド（Vue.js）
│   ├── pages/       ← 画面
│   ├── components/  ← UIパーツ
│   └── utils/       ← API通信
└── dist/            ← ビルド成果物（mainブランチで管理）
```

## 技術スタック

- **フロントエンド:** Vue.js 3 / Vue Router / Vite
- **バックエンド:** Google Apps Script
- **データ:** Google Spreadsheet（DB代わり）
- **メール解析:** Gmail API（Vpass通知メール）
- **ホスティング:** GitHub Pages（mainブランチ）

## セットアップ

### 1. バックエンド（GAS）

1. Googleスプレッドシートを新規作成
2. 拡張機能 > Apps Script を開く
3. `gas/api.gs` の内容を貼り付け
4. `SPREADSHEET_ID` を自分のスプレッドシートのURLに書き換え
5. `onOpen()` を一度実行（シートが自動作成される）
6. デプロイ > 新しいデプロイ > ウェブアプリ（全員アクセス可）

### 2. フロントエンド（ローカル確認）

```bash
npm install
npm run dev
```

ブラウザでダッシュボードを開き、API設定ボタンからデプロイしたGASのURLを入力してください。

### 3. GitHub Pagesへのデプロイ

GitHub Pagesは **mainブランチ** を使用しています。

```bash
npm run build
git add dist src
git commit -m "build: 説明"
git push origin main
```

GASのコードを変更した場合は、GASエディタにコピーして「デプロイを管理」→ 既存のデプロイを「新しいバージョン」で更新してください。

## アプリの使い方

### ホーム画面

| カード | 内容 |
|---|---|
| 予定 | 今日・明日のGoogleカレンダー予定 |
| ルーティン | 毎日のルーティンタスク（完了チェック可） |
| タスク | 自由記述のタスク管理（追加・完了・削除） |
| 今月の支払い見込み | クレカ＋口座引落の合計見込み金額 |
| 予算 | カテゴリ別の今月の予算と実績 |

### クレカ履歴画面

- 年月を選んでクレカ利用履歴を確認
- カテゴリ別の集計グラフ
- 今月の支払い見込み詳細（固定費・変動費の内訳）

### 冷蔵庫画面

- 食材の在庫管理（追加・更新）
- 在庫食材から作れるレシピ一覧

## スプレッドシートのシート構成

| シート名 | 用途 |
|---|---|
| VpassUsage | クレカ利用履歴（メールから自動取得） |
| MerchantMap | 利用先 → カテゴリのマッピング |
| Subscriptions | サブスク・固定費・借金の管理 |
| Budgets | カテゴリ別予算 |
| Todos | タスク管理 |
| Routines | ルーティンタスク |
| Fridge | 冷蔵庫在庫 |
| Recipes | レシピ |

### Subscriptionsシートの列構成

| 列 | 内容 | 例 |
|---|---|---|
| A | 名前 | Netflix |
| B | 金額（円） | 1490 |
| C | カテゴリ | エンタメ |
| D | 種別 | サブスク / 固定費 / 借金 |
| E | 支払い方法 | クレカ / 口座引落 |
| F | 頻度 | 毎月 / 年1 |
| G | 有効 | TRUE / FALSE |
| H | 開始月 | 2024-01 |
| I | 終了月 | （空欄で無期限） |
| J | 請求月 | 年1払いの請求月（1〜12） |
| K | 元本 | （借金の場合） |
| L | 残高 | （借金の場合） |
| M | 加盟店名 | ＮＦ＊ＮＥＴＦＬＩＸ |

**M列（加盟店名）について:**
VpassUsageの `merchant` 列に表示される文字列（部分一致）を入力すると、実際に請求された際に「請求済 ✓」として表示されます。空欄の場合は常に未請求として表示されます。
